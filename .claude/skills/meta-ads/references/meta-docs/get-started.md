# Get Started with the Marketing API

Updated: Jun 16, 2026

To use the Marketing API, you must set up your environment and gain access to the API's features.

## Ad account requirements

You must have an active ad account to run campaigns and manage billing. An ad account lets you track advertising expenses, monitor performance, and optimize campaigns.

### Finding your Ad Account number

Find it in Meta Ads Manager: log into Facebook, access Ads Manager, click ad account Settings (bottom left), and view the ad account number along with billing info and spending limits.

## Authorization and authentication

- Authorization — verifying users/apps accessing the Marketing API and granting permissions.
- Authentication — getting, extending, and renewing access tokens.
