# API Overview

The guides in this series provide an overview of the objects, methods, and
services available in the Google Ads API. After reading these guides, you will
understand the following key concepts:

1. [How the Google Ads API is structured](https://developers.google.com/google-ads/api/docs/concepts/api-structure).
2. [How Google Ads API versioning works](https://developers.google.com/google-ads/api/docs/concepts/versioning).
3. [Which API service to use to modify an object](https://developers.google.com/google-ads/api/docs/concepts/changing-objects).
4. [Which API service to use to retrieve an object and its performance statistics](https://developers.google.com/google-ads/api/docs/concepts/retrieving-objects).
5. [Which API service to use to retrieve API metadata](https://developers.google.com/google-ads/api/docs/concepts/field-service).
6. [How to structure API calls](https://developers.google.com/google-ads/api/docs/concepts/call-structure).
7. [How to mutate resources](https://developers.google.com/google-ads/api/docs/mutating/overview).
